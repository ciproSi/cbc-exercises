VSC shorcuts

  • CMD + K + C - comment a block
  • CMD + K + U - uncomment
  • CMD + K, CMD + C - comment line
  • SHIFT + ALT + arrow up/down - copy line up/down
  • CMD + SHIFT + V - HTML preview
  • ALT + W - wrap text into a tag
  • ALT + Z - text wrap
  • CMD + click - putting multiple cursors
